### Techniques & Tips

### Cascading
- **CSS Font Cascading**: Use `unicode-range` for cascading styles.
  - People can use `unicode-range` to do cascading.

  - Reference: Search for "Font Review BuckeyeCTF 2023"

  - Reference: Search for "STTF CSS trick"

### NoSQL Injection

### JWT (JSON Web Tokens)

  - Reference: Search for "certs BuckeyeCTF 2023 JWT RS256 to HS256"

### JSDom jail
- **Optimized Admin Bot vsCTF 2023**: Use `spawnSync` for RCE (requires `process` and `spawn_sync`).
  - Reference: Search for "Optimized Admin Bot vsCTF 2023 JSDom"

### CORS bypass
- **makes-sense ASIS-ctf-qual 2023**: Use shadow DOM to let an iframe access parent's document (`shadowContainer.attachShadow`).
  - Goal: Make `e.source` in `window.onmessage = e=>(e.source == top && e.source.length == 0 ? eval(e.data) : '')` not undefined.
  - Reference: Search for "makes-sense ASIS-ctf-qual 2023 CORS"

### CSP bypass
- **hacktricks.xyz**: Check this website for more CSP bypass tricks.
- **JS polyglots**: Embed JavaScript code in multiple language image files. Then use `<script charset="ISO-8859-1" src=url></script>` to execute.
  - References:
    - [JS JPEG polyglot script](https://portswigger.net/research/bypassing-csp-using-polyglot-jpegs)
    - [Bypassing CSP using polyglot JPEGs](https://portswigger.net/research/bypassing-csp-using-polyglot-jpegs) (Note: link seems duplicated in source, linking once)
- **CSS side-channel**: Using `color-mix` to make a tab crash with a signal.
  - References:
    - [Issue report](https://bugs.chromium.org/p/chromium/issues/detail?id=1490504) (example: `color-mix(in srgb, CanvasText 0%, transparent)` can crash tabs)
    - [Another-csp DiceCTF-2024-qual](https://github.com/dicegang/dicectf-quals-2024-challenges/tree/main/web/another-csp)

### JavaScript Jail
- **Inside of function**: Use `var process=this.constructor.constructor("return process;")();` to get the `process` object.
  - Then use `var require = global.require || global.process.mainModule.constructor._load;` or `this.require` to get RCE.
  - Example script: `spawnSync.js`

### Techniques & Tips
#### ROP
- If you pop a non-privileged shell using `system()` or similar from a SUID binary, you can do a `setuid(0)` syscall beforehand to get a privileged shell.
- `__nptl_change_stack_perm` can change stack NX permission.

#### Leak address
- **Leak libc, ld, stack, vdso, vvar addresses by an arbitrary read**: Sailing the C BuckeyeCTF-2024.
  - Reference: Search for "Sailing the C BuckeyeCTF-2024"

#### Disable ASLR temporary
- `setarch \`uname -m\` -R [executable]` from StackOverflow.

## Python Jail / Pyjail

- **starship CrewCTF 2023**: Character limit: `@^_":,.` and ASCII letters.
  - Payload: `type: @__build_class__.__self__.exec\r@__build_class__.__self__.input\rclass\x0cx:pass.`
- **Bonus_The_Revenge_of_Checkpass_1 BxMCTF 2023**: `eval(inp, {'__builtins__': None}, None)`.
  - Payload: `type: ().__class__.__bases__[0].__subclasses__().`
- **Username_Decorator BxMCTF 2023**: Flask escape.
  - Payload: `type: ().__class__.__bases__[0].__subclasses__().`
- **Sneckos-lair laCTF 2025**: Obtain function with `__code__` attribute by type alias without ability to define function and access to builtins.
  - Reference: Search for "Sneckos-lair laCTF 2025 pyjail"
- `'f{32:c}'`: Can format an int to char (equivalent to `chr(32)`).
- **Use other fonts to bypass alphabet filter**: E.g., use Italic alphabet from ItalicTextGenerator and use long underline like `＿𝘤𝘭𝘢𝘴𝘴＿`.
  - Example: Zero uoftctf-2024.
- **Boolean short circuit for side-channel**: `<` and `>` can short circuit.
  - `"Aa" > "Aa" > 0` will be `False`.
  - `"Ab" > "Aa" > 0` will raise a `TypeError` (so do other comparators).
- **PWN pyjail**:
  - Use `ctypes` to override the definition of functions (e.g., `os._exit`).
    - Example: diligent-auditor DiceCTF-2024-qual.
  - Use this behavior to do PWN.
    - Example: IRS DiceCTF-2024-qual.
  - Use UAF in fields like `issues#91153` audit hook POC.
    - Reference: [Python Issues #91153](https://github.com/python/cpython/issues/91153)
  - Use `vars()` and `|=` to add functions into globals (concat two dictionaries): `x=vars();x|=vars(list)`.
  - Use `match` to get member method or properties (PEP 636): `case dict(items=items1): print(items1())`.
  - Use `gc` module to get locals inside a function or deleted objects.
    - Example: `import gc; [obj for obj in gc.get_objects() if isinstance(obj, type_you_are_looking_for)]`
