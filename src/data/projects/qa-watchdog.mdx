# QA WatchDog

QA WatchDog is a Python-based tool created by Neeraj Singh, designed to streamline and simplify the QA testing process. With a strong focus on user-friendliness and collaboration, this tool empowers QA testers to efficiently track progress and manage their testing checklists.

## Key Features

### 1. User Login
![QA WatchDog Login](~/assets/images/qa-watchdog-login.png)

Collaboration is at the heart of QA testing, and QA WatchDog understands that. It allows multiple users to log in simultaneously, ensuring seamless teamwork.

### 2. Simple Installation
Gone are the days of complex installations. QA WatchDog offers a hassle-free installation process. You can download the setup file from the Releases page and get started in no time.

### 3. Organize with Multiple Checklists
Say goodbye to scattered checklists. QA WatchDog enables you to manage all your QA test checklists conveniently within a single account, keeping everything organized and easily accessible.

## Getting Started

1. **Download:** Visit the Releases page to download the setup file.
2. **Install:** Run the downloaded file to breeze through the installation process.
3. **Launch:** Open the app, create your account, and you're ready to dive in!
4. **Explore:** Start crafting checklists and experience smoother QA testing.
