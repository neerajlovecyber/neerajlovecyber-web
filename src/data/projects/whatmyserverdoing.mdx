# WhatMyServerDoing

WhatMyServerDoing is a server monitoring tool designed to provide real-time insights into server activity and performance. It helps administrators keep track of resource usage, running processes, and system health.

## Key Features

- Real-time monitoring of CPU, memory, and disk usage
- Process tracking and alerts
- User-friendly dashboard
- Easy setup and configuration

## Getting Started

1. **Download:** Get the latest release from the Releases page.
2. **Install:** Follow the installation instructions in the README.
3. **Launch:** Start monitoring your server instantly.
